NAME Cerium
SYMBOL Ce
ATOMIC NUMBER 58
APPEARANCE Silvery white
CLASSIFICATION Lanthanide
BOILING POINT 3716 K
MELTING POINT 1068 K
DESCRIPTION
  • Cerium is the second element of the lanthanide series.
  • It is soft enough to be cut with a knife.
  • It often shows the +3 oxidation state characteristic of the series.
  • it also exceptionally has a stable +4 state.
