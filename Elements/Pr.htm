NAME Praseodymium
SYMBOL Pr
ATOMIC NUMBER 59
APPEARANCE Grayish white
CLASSIFICATION Lanthanide
BOILING POINT 3403 K
MELTING POINT 1208 K
DESCRIPTION
  • It most readily forms the +3 oxidation state, which is the only stable state in aqueous solution.
  • Although the +4 oxidation state is known in some solid compounds.
  • The +5 oxidation state is attainable in matrix-isolation conditions.
  • Aqueous praseodymium ions are yellowish-green
