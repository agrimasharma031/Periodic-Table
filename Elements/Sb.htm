NAME Antimony
SYMBOL Sb
ATOMIC NUMBER 51
APPEARANCE Silvery lustrous gray
CLASSIFICATION Metalloid
BOILING POINT 1908 K
MELTING POINT 903.78 K
DESCRIPTION
  • Antimony is a member of group 15 of the periodic table.
  • It is one of the elements called pnictogens.
  • It has an electronegativity of 2.05.
