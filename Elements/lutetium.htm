NAME Lutetium
SYMBOL Lu
ATOMIC NUMBER 71
APPEARANCE silvery-white.
CLASSIFICATION Metallic
BOILING POINT 3402 C
MELTING POINT 1663 C
DESCRIPTION
  • It is a silvery white metal, which resists corrosion in dry air, but not in moist air.
  • Lutetium is the last element in the lanthanide series, and it is traditionally counted among the rare earths.
  • Lutetium oxide is used to make catalysts for cracking hydrocarbons in the petrochemical industry.
