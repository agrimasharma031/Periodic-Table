NAME Osmium
SYMBOL Os
ATOMIC NUMBER 76
APPEARANCE hard, brittle, bluish-white.
CLASSIFICATION Metallic
BOILING POINT 5030 C
MELTING POINT 3045 C
DESCRIPTION
  • It is a hard, brittle, bluish-white transition metal
  • found as a trace element in alloys, mostly in platinum ores.
  • It is used to produce very hard alloys for fountain pen tips, instrument pivots, needles and electrical contacts.
