NAME Rhenium
SYMBOL Re
ATOMIC NUMBER 75
APPEARANCE silvery-gray, heavy.
CLASSIFICATION Metallic
BOILING POINT 5600 C
MELTING POINT 3180 C
DESCRIPTION
  • It is a silvery-gray, heavy, third-row transition metal in group 7 of the periodic table.
  • With an estimated average concentration of 1 part per billion (ppb), rhenium is one of the rarest elements in the Earth's crust.
  • Rhenium is used with platinum as catalysts in the production of lead-free, high-octane gasoline.
