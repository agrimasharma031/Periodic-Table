NAME Tantalum
SYMBOL Ta
ATOMIC NUMBER 73
APPEARANCE hard, blue-gray, lustrous.
CLASSIFICATION Metallic
BOILING POINT 5458 C
MELTING POINT 3017 C
DESCRIPTION
  • Previously known as tantalium, it is named after Tantalus, a villain from Greek mythology.
  • Tantalum is a rare, hard, blue-gray, lustrous transition metal that is highly corrosion-resistant.
  • Tantalum is used in the electronics industry for capacitors and high power resistors.
