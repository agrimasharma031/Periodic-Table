NAME Tin
SYMBOL Sn
ATOMIC NUMBER 50
APPEARANCE Silvery-white (beta, β) or gray (alpha, α)
CLASSIFICATION Post Transition Metal
BOILING POINT 2875 K
MELTING POINT 505.08 K
DESCRIPTION
  • Tin is a silvery metal.
  • Characteristically it has a faint yellow hue.
  • Tin, like indium, is soft enough to be cut without much force
  • The first tin alloy used on a large scale was bronze.
