NAME Tennesine

CLASSIFICATION Unknown
BOILING POINT 883 K

DESCRIPTION
  • Tennessine is a radioactive, artificially produced element.
  • It is expected to be a solid, but its classification is unknown.
  • It is a member of the halogen group.
  • The element, No. 117 on the Periodic Table of Elements, had previously been designated ununseptium,
    a placeholder name that means one-one-seven in Latin.
