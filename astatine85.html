NAME Astatine
SYMBOL At
ATOMIC NUMBER 85
APPEARANCE unknown, probably metallic
CLASSIFICATION Metalloid
BOILING POINT 503 K
MELTING POINT - K
DESCRIPTION
  • It is the rarest naturally occuring element in earth's crust.
  • A sample of the pure element has never been assembled, because any macroscopic specimen would be immediately vapourized by its own radioactivity.
  • The bulk properties of astatine are not known with any certainity.
