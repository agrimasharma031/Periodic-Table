NAME Bismuth
SYMBOL Bi
ATOMIC NUMBER 83
APPEARANCE lustrous brownish silver
CLASSIFICATION Metallic
BOILING POINT 1837 K
MELTING POINT 544.7 K
DESCRIPTION
  • It is a brittle metal with silver white color when freshly produced, but oxidation can give it a pink tinge.
  • Bismuth is the most naturally dimagnetic element, and has one of the lowest values of thermal conductivity amoung metals.
  • the free element is 86% as dense as Lead.
