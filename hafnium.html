NAME Hafnium
SYMBOL Hf
ATOMIC NUMBER 72
APPEARANCE lustrous, silvery gray.
CLASSIFICATION Metallic
BOILING POINT 4600 C
MELTING POINT 2227 C
DESCRIPTION
  • A lustrous, silvery gray, tetravalent transition metal.
  • hafnium chemically resembles zirconium and is found in many zirconium minerals.
  • Hafnium is a good absorber of neutrons and is used in the control rods of nuclear reactors.
