NAME Mercury
SYMBOL Hg
ATOMIC NUMBER 80
APPEARANCE dense, silvery-white, poisonous metal with a mirror-like appearance
CLASSIFICATION Metallic
BOILING POINT 357 C
MELTING POINT -38.9 C
DESCRIPTION
  • Mercury is a dense, silvery-white, poisonous metal with a mirror-like appearance.
  • It is the only common metal that is liquid at room temperature.
  • Mercury evaporates slowly at room temperature.
