NAME Oganesson
SYMBOL Og
ATOMIC NUMBER 118
APPEARANCE Unknown, but probably a colouress gas
CLASSIFICATION Non Metallic
BOILING POINT Unknown
MELTING POINT Unknown
DESCRIPTION
  • Oganesson is a radioactive, artificially produced element about which little is known.
  • It is expected to be a gas and is classified as a non-metal.
  • It is a member of the noble gas group.
  • It was recognized as one of four new elements by the Joint Working Party of the international
    scientific bodies IUPAC and IUPAP.
