NAME Platinum
SYMBOL Pt
ATOMIC NUMBER 78
APPEARANCE malleable, ductile, highly unreactive, precious, silverish-white
CLASSIFICATION Metallic
BOILING POINT 3827 C
MELTING POINT 1774 C
DESCRIPTION
  • It is a dense, malleable, ductile, highly unreactive, precious, silverish-white transition metal.
  • Its name is derived from the Spanish term platino, meaning "little silver".
  • The metal has excellent resistance to corrosion, is stable at high temperatures and has stable electrical properties.
