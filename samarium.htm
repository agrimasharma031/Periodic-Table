NAME Samarium
SYMBOL Sm
ATOMIC NUMBER 62
APPEARANCE bright, fairly hard, silvery white metal.
CLASSIFICATION Metallic
BOILING POINT 1794 C
MELTING POINT 1072 C
DESCRIPTION
  • Samarium was discovered in 1879 by the French chemist Paul-Émile Lecoq
    de Boisbaudran and named after the mineral samarskite from which it was isolated.
  • The major commercial application of samarium is in samarium–cobalt magnets.
  • Although classified as a rare-earth element, samarium is the 40th most abundant
    element in the Earth's crust
  • Samarium has no significant biological role but is only slightly toxic.
