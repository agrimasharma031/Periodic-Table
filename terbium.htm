NAME Terbium
SYMBOL Tb
ATOMIC NUMBER 65
APPEARANCE silvery-white and soft enough to be cut with a knife.
CLASSIFICATION Metallic
BOILING POINT 3227 C
MELTING POINT 1356 C
DESCRIPTION
  • It is a silvery-white, rare earth metal that is malleable, ductile, and soft enough to be cut with a knife.
  • The ninth member of the lanthanide series, terbium is a fairly electropositive metal
  • reacts with water, evolving hydrogen gas.
