NAME Thallium
SYMBOL Tl
ATOMIC NUMBER 81
APPEARANCE grey colored
CLASSIFICATION Metallic
BOILING POINT 1746 K
MELTING POINT 577 K
DESCRIPTION
  • Thallium is a gray post transition metal that is found free in nature.
  • when isolated, thallium resembles tin, but discolors when exposed to air.
  • chemist William Crookes and Claude Auguste Lamy discovered thalium independently in 1861.
