NAME Tungsten
SYMBOL W
ATOMIC NUMBER 74
APPEARANCE dull silver-colored.
CLASSIFICATION Metallic
BOILING POINT 5555 C
MELTING POINT 3422 C
DESCRIPTION
  • Tungsten is a dull silver-colored metal with the highest melting point of any pure metal.
  • tungsten is more resistant to fracturing than diamond and is much harder than steel.
  • Tungsten is commonly used in heavy metal alloys such as high speed steel, from which cutting tools are manufactured.
